Players.csv

  • This is a csv file of all the NBA players that have played in the NBA. In this file the data I am going to use is the player names, their height and weight, born and the college they went too. The database I will need will use this data will be a database named Players, with the columns of player_name, height, weight, born and college. I plan to use this data so you can easily look up which college players have gone to, as well as be able to see who were the tallest/shortest and heaviest/lightest players in the NBA.
  • Reference: https://www.kaggle.com/drgilermo/nba-players-stats/data
